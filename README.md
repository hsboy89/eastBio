# 이스트 바이오 공식 웹사이트

이스트 바이오(East Bio) 브랜드 랜딩 페이지입니다.

## 프로젝트 개요

의약품 도매업의 전문성을 시각적으로 강조하고, 파트너사(제약사/의료기관)에게 현대적이고 신뢰할 수 있는 기업 이미지를 전달하는 원페이지 웹사이트입니다.

## 기술 스택

- **React 19** + **TypeScript**
- **Vite** - 빌드 도구
- **Framer Motion** - 애니메이션 라이브러리
- **Lenis** - 스무스 스크롤
- **React Helmet Async** - SEO 최적화
- **Pretendard** - 웹폰트

## 주요 기능

- 🎨 **고급스러운 애니메이션**: Framer Motion을 활용한 스크롤 기반 애니메이션
- 📱 **완전 반응형**: PC, 태블릿, 모바일 최적화
- 🎯 **원페이지 구조**: 스크롤을 통한 모든 정보 접근
- ⚡ **빠른 로딩**: Vite 기반 최적화된 빌드
- 🔍 **SEO 최적화**: React Helmet을 통한 메타 태그 관리

## 섹션 구성

1. **Hero Section**: 강렬한 비주얼과 핵심 슬로건
2. **Company Intro**: 기업 철학 및 대표자 인사말
3. **Business Area**: 전문의약품, 일반의약품, 의료기기 유통 서비스 소개
4. **Strengths**: 콜드체인 시스템, 신속 배송 네트워크 등 기술적 강점
5. **Partners & Network**: 협력 제약사 및 공급망 현황
6. **Contact**: 오시는 길, 문의처 정보

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 배포

이 프로젝트는 정적 사이트이므로 다음 플랫폼에 배포할 수 있습니다:

- **Vercel**: `vercel deploy`
- **Netlify**: Netlify에 프로젝트 연결
- **GitHub Pages**: `npm run build` 후 `dist` 폴더 배포
